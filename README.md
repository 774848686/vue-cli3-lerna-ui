# vue-cli3-lerna-ui

## Project setup

``` bash
npm i yarn lerna -g
git clone https://github.com/ziyi2/vue-cli3-lerna-ui.git
```

### Compiles and hot-reloads for development

``` bash
npm run serve
```

### Compiles and minifies for production

``` bash
npm run build
```

### Lints and fixes files

``` bash
npm run lint
```
